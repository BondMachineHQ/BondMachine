# cmpv

**Instruction**: cmpv

**Length**: opBits + inBits

**Description**:

The CMPV instruction check if the input given as argument has a valid signal and sets the cmpflag accordingly. If the input has valid data, cmpflag is set to 1, otherwise it is set to 0. The flag cmpflag can be used by other instructions to take decisions based on the result of this comparison.
